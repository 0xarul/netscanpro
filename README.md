# NetScan Pro

Professional Network & Web Security Assessment Framework written in Python.

## Features

- Multi-threaded Port Scanner
- Banner Grabbing
- Web Vulnerability Assessment
- Security Header Analysis
- SQLi & XSS Detection Indicators
- Sensitive Path Discovery
- HTML Report Generation
- Risk Classification

## Installation

```bash
git clone https://github.com/0xarul/NetScan-Pro.git
cd NetScan-Pro
pip install -r requirements.txt
```

## Usage

### Port Scan

```bash
python3 netscanpro.py --target 127.0.0.1 --ports
```

### Full Scan

```bash
python3 netscanpro.py --target 127.0.0.1 --full
```
